# Gunpo Siren Order (BizCool Order)

## 제작 배경
교내에서 진행하는 경진대회에 출품할 목적으로 만든 오더 플랫폼입니다.

react native를 사용하여 제작하였으며, react-redux를 활용한 상태관리 기능을 추가하였습니다.

스타벅스를 포함한 여러 어플리케이션의 디자인을 참고하였고, 컴포넌트의 디자인은 eva design의 라이브러리를 활용하였습니다.
